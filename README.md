## Running the project

### Run Expo Environment

```
npx expo start / yarn start -c
```

### Run Expo Environment

### To Test with DeepSeek IA

```
install ollama
ollama run deepseek-r1:1.5b
```

**Install dependencies**

```
npm / yarn
```

**Install IOS Pods**

```
cd ios && pod install
```

**Run IOS**

```
yarn ios
```

**Run Android**

```
yarn android
```

## Running the tests

Use **yarn test** to execute the jest tests.

```cl
yarn test
```
